Throwing and returning

Some other details
  • return is used to set the value when {{ '{' }} done: true }
  • The return value, however, isn't included when iterating using spread, or for...of
  • Throwing an exception inside a generator will return that error on next()
  • ...and will complete the generator
function* myGenerator() {{ '{' }}
  throw new Error('Oops...');
}

      
const generator = myGenerator();

      
generator.next(); // Uncaught Error: Oops...

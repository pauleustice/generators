Intro to Generators

Overview
  • A hybrid between a regular function and an Iterable.
  • They stop and start, and can keep track of internal state.
  • A generator yields multiple values across runs
Key details
  • Differentiated from normal functions using function* myGenerator() asterisk syntax
  • When called, returns a reference to a Generator Object, upon which you call next()
  • next() returns an object with two keys: value and done

Yielding values

A basic example
  • Calling next() yields the next value within the generator
function *numbersGenerator() {{ '{' }}
  yield 3;
  yield 2;
  yield 1;
}

      
const countdown = numbersGenerator();

      
countdown.next();
countdown.next();
countdown.next();
countdown.next();
 
Yielded value:
 
// {{ '{' }} value: 3, done: false }
// {{ '{' }} value: 2, done: false }
// {{ '{' }} value: 1, done: false }
// {{ '{' }} value: undefined, done: true }

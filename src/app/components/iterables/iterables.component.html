A quick overview of iterables

Key details
  • Native iterables are things like array, string, map, set
  • They provide an interface within Symbol.iterator that defines how their contents are iterated
const example = [ 1, 2, 3 ];
const exampleIterator = example[Symbol.iterator]();

      
exampleIterator.next();
// {{ '{' }} value: 1, done: false }
exampleIterator.next();
// {{ '{' }} value: 2, done: false }
exampleIterator.next();
// {{ '{' }} value: 3, done: false }
exampleIterator.next();
// {{ '{' }} value: undefined, done: true }
  • So, Symbol.iterator provides the internal workings of how to iterate the content
  • ...and it provides the next function which provides the next value, until the end of the data
